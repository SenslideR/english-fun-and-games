# Fun and games — English learning app

Интерактивное приложение для изучения английского на основе раздела учебника
**Unit 3 «Fun and games»**: карточки для запоминания слов и интерактивные упражнения.

## Возможности

- 📚 **Карточки (flashcards)** — словарь, фразовые глаголы и предложные сочетания.
  Карточка переворачивается (перевод + определение + пример), есть самооценка
  «Знаю / Повторить» и простое интервальное повторение (5 уровней запоминания).
- ✏️ **Интерактивные упражнения** — выбор варианта и вставка слова, с мгновенной
  проверкой ✓/✗, подсказками и подсчётом результата.
- 🎯 **Тренировка слов** — три режима, сгенерированные из словаря: аудирование
  (🔊 → ввод), квиз перевода (RU ↔ EN) и сопоставление слово ↔ перевод.
- 🧩 **Кроссворд** — сцепленная сетка с вводом букв, навигацией стрелками,
  проверкой (зелёный/красный) и кнопкой «Показать ответы».
- 🔊 **Озвучка** — кнопка-динамик произносит слова, примеры и слова из банка
  (Web Speech API, английский голос браузера; без внешних сервисов).
- 💾 **Прогресс** сохраняется в браузере (`localStorage`).

## Запуск

```bash
npm install     # один раз
npm run dev     # http://localhost:5173
```

Сборка продакшн-версии:

```bash
npm run build   # в папку dist/
npm run preview
```

## Как добавить новый раздел учебника

Весь контент вынесен в `src/data/unit3.ts` (типы — в `src/types.ts`).
Чтобы добавить новый юнит: создайте файл по образцу `unit3.ts`, заполните
`decks` (карточки) и `exercises` (упражнения) и подключите его в `src/App.tsx`.

Типы упражнений:
- `kind: 'choice'` — выбор из вариантов (`options`), например multiple choice;
- `kind: 'input'` — ввод слова (`answers` — список принимаемых ответов).
